Why This Website Exists

When I started my journey to become a Java backend developer six months ago, I found a plenty of roadmaps for frontend such as JavaScript, but the resources for aspiring Java developers were sparse. This made the initiation process somewhat confusing and challenging for many beginners, including me.

Even though I've been learning Java for just half a year and am still navigating my way through, I decided to create this website. It is a reflection of the roadmap I'm following in my learning process. It might not be exhaustive, but my hope is that it can be a helpful starting point for others on a similar journey.

Please bear in mind that I'm still learning. If you find any errors or have suggestions to improve this roadmap, feel free to contact me!

How To Use This Roadmap

Firstly, I recommend starting with the "Overview of Programming" section. This segment provides valuable insights into a programmer's role and the mindset that can contribute to your success.

Next, dive into the HTML course. As you progress, you'll likely face challenges; there might be times when things don't go as planned, and you may feel despondent. However, concurrently, you'll also experience the thrill of building a webpage, which can be very rewarding.

While taking the HTML course, I also suggest you go through the "CS Crash Course". It is beginner-friendly and elucidates various important concepts which will be beneficial throughout your career as a developer.

Moving on, you should take the JAVA,CS50,Object Oriented Programming together, and then simultaneously study MySQL, NAND2TETRIS, DSA, and Clean Code and so on. But Remember, this roadmap is flexible. If you find certain topics in the "Computer Science" or "Additional Resources" section challenging, or if your primary goal is to quickly learn Java & Spring to get a job, you can revisit these complex topics later when you're comfortable or when the need arises, And you can also study those subjects individually.

But you should study Language part not put off even if it's hard.

This roadmap is designed to be a living resource, evolving with your learning journey. So, let's start this exciting adventure of becoming a Java backend developer!
